# Box

---

盒子样式，主要用来包含其他的元素，其主要包含以下属性 `shadow`, `border`, `radius`, `padding`

## 基础盒子

:::demo

```html
<template>
  <div class="ufin-box">
    默认的.panel所做的只是提供基本的边界和内部，来包含内容。
  </div>
</template>
```

:::

## a 标签

:::demo

```html
<template>
  <a class="ufin-box" href="#">
    这是一个a标签
  </a>
</template>
```

:::
